## FAQ Quick Answers

**Q: Do I need to install anything?**
A: No! Use the web app or bookmarklet directly. Just open in a browser.

**Q: Will my reading progress sync across devices?**
A: No, it's stored locally in your browser. Use the same browser/device.

**Q: Can I modify the colors/styling?**
A: Yes! See [Customization > Change Colors](./implementation.md#1-change-colors--styling)

**Q: How accurate is thread detection?**
A: 85-95% for well-formatted comments. See [Accuracy](./algorithms.md#accuracy--edge-cases)

**Q: Can you support my blog platform?**
A: Probably! File an issue with your blog URL and theme name.

**Q: Is my data private?**
A: Yes, completely. See [Privacy](./quickstart.md#data-privacy)
